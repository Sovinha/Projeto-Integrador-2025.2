1. DESCRIÇÃO DO PROJETO
--------------------------------------------------------
O projeto "Portal Acadêmico 2025.2" foi desenvolvido como
atividade prática de desenvolvimento web para o projeto integrador.

O objetivo é simular um pequeno sistema acadêmico com:
- Tela de login do aluno;
- Tela de digitação de notas (Nome, RGM, Turma, Nota 1, Nota 2).

O projeto foi dividido em duas etapas:
1) Criação dos WIREFRAMES (protótipos em imagem);
2) Construção do FRONT-END utilizando o framework Bootstrap.

3. TECNOLOGIAS UTILIZADAS
--------------------------------------------------------
- HTML5
- CSS3
- Bootstrap 5 (versão local, sem CDN)
- JavaScript (para redirecionar usuário entre as telas)

--------------------------------------------------------
4. RECURSOS DO BOOTSTRAP UTILIZADOS
--------------------------------------------------------
- Sistema de Grid (row e col)
- Input Groups
- Buttons
- Containers e Cards para layout
- Responsividade automática

--------------------------------------------------------
5. FUNCIONALIDADES
--------------------------------------------------------
- Tela de login com campos de Usuário e Senha;
- Redirecionamento para a tela de notas;
- Saudação automática com o nome digitado no login;
- Tela de notas com campos de Nome, RGM, Turma, Nota 1 e Nota 2;
- Botões “Salvar” e “Limpar” funcionais (HTML padrão);
- Botão “Sair” na tela de notas para voltar ao login.

--------------------------------------------------------
6. COMO EXECUTAR O PROJETO
--------------------------------------------------------
1. Baixe o arquivo ZIP do projeto.
2. Extraia a pasta em seu computador.
3. Abra o arquivo "index.html" em um navegador (Chrome, Edge, etc.).
4. Digite um nome e senha para fazer login.
5. O sistema redirecionará para a tela de notas.
6. Clique em “Sair” para retornar ao login.
